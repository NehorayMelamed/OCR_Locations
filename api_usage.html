API Documentation

1. Process Image

Endpoint to process an image and extract text using OCR.

Endpoint: /process_image
Method: POST
Request Body:
{
    "image_data": "base64_encoded_image_or_path"
}
Response:
{
    "status": "success",
    "message": "Image processed successfully!",
    "results": {
        "lines": [
            {
                "text": "extracted_text",
                "confidence": confidence_value,
                "bounding_box": bounding_box_coordinates
            },
            ...
        ]
    }
}

2. Process Video

Endpoint to process a video and extract text from frames using OCR. Videos longer than the defined MAX_VIDEO_DURATION (default 600 seconds) will be rejected.

Endpoint: /process_video
Method: POST
Request Body:
{
    "video_data": "base64_encoded_video_or_path",
    "frame_stride": optional_frame_stride_value,
    "num_of_cores": optional_number_of_cores,
    "return_base64": optional_return_base64_flag
}
Response:
{
    "status": "success",
    "message": "Video processed successfully!",
    "results": [
        {
            "frame_number": frame_number,
            "frame_time": frame_time_in_seconds,
            "lines": [
                {
                    "text": "extracted_text",
                    "confidence": confidence_value,
                    "bounding_box": bounding_box_coordinates
                },
                ...
            ]
        },
        ...
    ]
}

3. Is Alive

Endpoint to check if the service is running and alive.

Endpoint: /is_alive
Method: GET
Response:
{
    "status": "success",
    "message": "Service is alive!"
}
